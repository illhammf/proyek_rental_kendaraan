# POSTER PROYEK

## SISTEM RENTAL KENDARAAN BERBASIS OBJECT-ORIENTED PROGRAMMING

### Masalah
Pencatatan rental kendaraan secara manual berisiko menimbulkan data ganda, kesalahan status kendaraan, dan laporan transaksi yang tidak rapi.

### Solusi
Aplikasi terminal berbasis Python untuk mengelola kendaraan, pelanggan, transaksi sewa, pengembalian, penyimpanan data, dan laporan.

### Fitur
- Data mobil dan motor
- Data pelanggan
- Transaksi sewa
- Pengembalian kendaraan
- Statistik rental
- Laporan TXT dan CSV
- Penyimpanan JSON

### Konsep OOP
- Class dan Object
- Inheritance: Mobil dan Motor mewarisi Kendaraan
- Encapsulation: atribut private dengan getter dan setter
- Polymorphism: hitung_biaya_sewa dioverride
- Composition: Rental memiliki FileManager dan Laporan

### Teknologi
Python, VS Code, JSON, Terminal CLI
