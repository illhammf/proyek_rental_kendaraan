# Cara Menjalankan di VS Code

1. Extract file ZIP.
2. Buka VS Code.
3. Klik File > Open Folder.
4. Pilih folder `Sistem_Rental_Kendaraan_OOP`.
5. Buka terminal VS Code.
6. Jalankan perintah:

```bash
python main.py
```

Untuk menjalankan testing:

```bash
python tests/test_rental.py
```

Jika memakai Windows dan `python` tidak terbaca, coba gunakan:

```bash
py main.py
py tests/test_rental.py
```
